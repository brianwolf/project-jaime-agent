# :card_index_dividers: project-jaime-agent

> Agente de Jaime para correr los scripts

![alt](img/logo.png)

## :gear: Requisitos

* python 3.11
* virtualenv

## :tada: Uso

```bash
# Levantar el ambiente
virtualenv -p python3.11 env
. env/bin/activate
pip install -r requirements.txt

# Ejecutar
python app.py
```
