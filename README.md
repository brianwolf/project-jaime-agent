# :card_index_dividers: project-jaime-agent

> Agente de Jaime para usar OC

![alt](img/logo.png)

## :gear: Requisitos

* python 3.10
* virtualenv

## :tada: Uso

```bash
# Levantar el ambiente
virtualenv -p python3.10 env
. env/bin/activate
pip install -r requirements.txt

# Ejecutar
python app.py
```
